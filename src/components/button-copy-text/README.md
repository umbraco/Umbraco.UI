# uui-button-copy-text

Umbraco style text-copy component.

### See it in action

Preview the component on [Storybook](https://uui.umbraco.com/?path=/docs/uui-button-copy-text--docs)

## Installation

```zsh
npm i @umbraco-ui/uui
```

Import the registration of `<uui-button-copy-text>` via:

```javascript
import '@umbraco-ui/uui/components/button-copy-text/button-copy-text.js';
```

When looking to leverage the `UUIButtonCopyTextElement` base class as a type and/or for extension purposes, do so via:

```javascript
import { UUIButtonCopyTextElement } from '@umbraco-ui/uui/components/button-copy-text/button-copy-text.js';
```

Alternatively, if you have already imported the full library, the element will be registered automatically:

```javascript
import '@umbraco-ui/uui';
```

## Usage

```html
<uui-button-copy-text></uui-button-copy-text>
```
