# uui-symbol-lock

Umbraco style symbol-lock component.

### See it in action

Preview the component on [Storybook](https://uui.umbraco.com/?path=/docs/uui-symbol-lock--docs)

## Installation

```zsh
npm i @umbraco-ui/uui
```

Import the registration of `<uui-symbol-lock>` via:

```javascript
import '@umbraco-ui/uui/components/symbol-lock/symbol-lock.js';
```

When looking to leverage the `UUISymbolLockElement` base class as a type and/or for extension purposes, do so via:

```javascript
import { UUISymbolLockElement } from '@umbraco-ui/uui/components/symbol-lock/symbol-lock.js';
```

Alternatively, if you have already imported the full library, the element will be registered automatically:

```javascript
import '@umbraco-ui/uui';
```

## Usage

```html
<uui-symbol-lock></uui-symbol-lock>
```
