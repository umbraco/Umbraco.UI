# uui-symbol-file-dropzone

Umbraco style symbol-file-dropzone component.

### See it in action

Preview the component on [Storybook](https://uui.umbraco.com/?path=/docs/uui-symbol-file-dropzone--docs)

## Installation

```zsh
npm i @umbraco-ui/uui
```

Import the registration of `<uui-symbol-file-dropzone>` via:

```javascript
import '@umbraco-ui/uui/components/symbol-file-dropzone/symbol-file-dropzone.js';
```

When looking to leverage the `UUISymbolFileDropzoneElement` base class as a type and/or for extension purposes, do so via:

```javascript
import { UUISymbolFileDropzoneElement } from '@umbraco-ui/uui/components/symbol-file-dropzone/symbol-file-dropzone.js';
```

Alternatively, if you have already imported the full library, the element will be registered automatically:

```javascript
import '@umbraco-ui/uui';
```

## Usage

```html
<uui-symbol-file-dropzone></uui-symbol-file-dropzone>
```
