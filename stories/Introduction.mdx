# Umbraco UI Library <small>v{UUI_VERSION}</small>

Welcome to the **Umbraco UI (UUI)** component library — a collection of 80+ reusable web components built with [Lit](https://lit.dev/).

## Installation

```bash
npm install @umbraco-ui/uui
```

## Usage

Import individual components for tree-shaking:

```js
import '@umbraco-ui/uui/components/button/button.js';
```

Or import everything:

```js
import '@umbraco-ui/uui';
```

Then choose a theme:

```html
<link rel="stylesheet" href="@umbraco-ui/uui/themes/light.css" />
<!-- or -->
<link rel="stylesheet" href="@umbraco-ui/uui/themes/dark.css" />
```
