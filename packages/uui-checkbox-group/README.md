# uui-checkbox-group

![npm](https://img.shields.io/npm/v/@umbraco-ui/uui-checkbox-group?logoColor=%231B264F)

Umbraco style checkbox-group component.

## Installation

### ES imports

```zsh
npm i @umbraco-ui/uui-checkbox-group
```

Import the registration of `<uui-checkbox-group>` via:

```javascript
import '@umbraco-ui/uui-checkbox-group';
```

When looking to leverage the `UUICheckboxGroupElement` base class as a type and/or for extension purposes, do so via:

```javascript
import { UUICheckboxGroupElement } from '@umbraco-ui/uui-checkbox-group';
```

## Usage

```html
<uui-checkbox-group></uui-checkbox-group>
```
